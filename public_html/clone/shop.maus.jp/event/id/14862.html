Maus™とは？

オレンジ色のカラダに茶色の耳、大きな目が愛らしい「Maus／マウス」は、1971年にドイツで放送開始したTV番組「Die Sendung mit der Maus（マウスといっしょ）」から生まれたキャラクター。
この番組は、「マウス」がゾウの「エレファント」とアヒルの「エンテ」と一緒に物の仕組みや色々な謎を解いて、わかりやすく紹介するエデュテイメント・アニメとして、40年以上前からドイツで最大の人気を誇る子供向け番組です。
世の中を面白い視点でとらえる構成は、大人でも発見することの多い面白さを盛り込んでいます。
